#  深入浅出node 第二章 笔记



### 概述

CommonJS中的大部分规范，为JavaScript开发大型应用程序指明了一条非常好的道路。

CommonJS规范涵盖了模块、二进制、Buffer、字符集编码、IO流、进程环境、文件系统、套接字、单元测试、web服务器网管接口、包管理等。

Node与浏览器，以及W3C组织、CommonJS组织、ECMAScript之间的关系，共同构成一个繁荣的生态系统。
